Onderdeel 1:

Intro

1. Wat is het verschil tussen het internet en het world wide web?

Het world wide web, of kortweg web, zijn de pagina's die je ziet als je op een apparaat zit en online bent. Maar het internet is het netwerk van verbonden computers waarop het web werkt, en waarover e-mails en bestanden reizen.


2. Wat is het verschil tussen HTML, CSS en JavaScript?

HTML geeft de structuur van een pagina aan. CSS zorgt voor de opmaak. Javascript zorgt voor hoe een pagina zich moet gedragen.


3. Wat is een HTML tag en een HTML element? Hoe open je een HTML tag en hoe sluit je een HTML tag. Maak hiervoor voorbeelden en leg uit.

Een HTML-element wordt gebruikt om structuur aan je website te geven en je content in te gieten. Hierdoor kun je je content op een bepaalde manier vertonen. Een HTML-element bestaat uit een opening -en sluitingtag. bijv. <p> Content </p>


4. HTML gebruikt basis tags om structuur op te bouwen. De <html>, <head> en <body>. Geef voor elke element aan waarvoor het dient.

Het <html> element is het root element en het definieert het hele HTML document.
De body omvat de visuele inhoud van de html-pagina.
De head bevat informatie die wordt meegegeven aan de browser.


5. Op HTML elementen kun je attributen gebruiken. Wat zijn HTML attributen ? Geef minimaal 5 attributen en maak voorbeelden waar je de attributen in toepast.

  • Alle HTML-elementen kunnen attributen bevatten
  • Attributen geven extra informatie over elementen
  • Attributen worden altijd gespecificeerd in de start tag
  • Attributen komen meestal in naam/waarde-paren zoals: name="value"

7. Wat is het verschil tussen inline en block elementen?

Elementen op blok-niveau

  • Een block-level element begint altijd op een nieuwe regel.
  • Een block-level element neemt altijd de volledige beschikbare breedte in beslag (strekt zich naar links en rechts zo ver uit als het kan).
  • Een block-level element heeft een boven- en ondermarge, terwijl een inline-element dat niet heeft.

Elementen op inline-niveau
  • Een inline-element begint niet op een nieuwe regel.
  • Een inline-element neemt slechts zoveel breedte in als nodig is. .
  • Dit is een <span> element binnen een alinea.

8. Wat betekent nesting? Codeer een voorbeeld met nesting erin.

Nesting is dat je een element onderbrengt in een ander element. Bijvoorbeeld, <h1> naam <span> ondertitel </span> </h1>
