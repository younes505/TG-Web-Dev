• ::backdrop, definieert de achtergrond van een element dat weergegeven wordt in fullscreen-mode.
  • ::cue, CSS pseudo-element past WebVTT bijschriften aan binnen een geselecteerd element. Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media met VTT-sporen te stijlen.
  • ::file-selector-button, vertegenwoordigt de knop van een van type="bestand".
  • ::marker, selecteert het markeringsvak van een lijst-item, dat meestal een opsommingsteken of nummer bevat. Het werkt op elk element of pseudo-element dat is ingesteld op weergave: list-item, zoals de <li>.
  • ::placeholder, staat voor de placeholder-tekst in een <input> of <textarea> element.
  • ::selection, past stijlen toe op het deel van een document dat door de gebruiker is gemarkeerd (zoals klikken en slepen met de muis over tekst).
