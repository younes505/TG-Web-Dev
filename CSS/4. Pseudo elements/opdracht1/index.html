• ::before, voegt een denkbeeldige element toe aan het begin van de inhoud van een regulier element.
  • ::after, voegt een denkbeeldige element toe aan het eind van de inhoud van een regulier element.
  • ::first-line, Hiermee kun je styling toevoegen aan de eerste letter van een block-level-element.
  • ::first-letter, Hiermee kun je styling toevoegen aan de eerste regel van een block-level-element.
