1. Er zijn verschillende selectors, bijvoorbeeld; type selectors, descendant selectors, class selectors. Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

  • type selectors: je selecteert hiermee het element dat je wilt ragetten. vb: p{}. Voordeel is dat je meerdere elementen kunt targetten met die naam.
  • descendant selectors: je geeft hiermee aan dat je een element selecteert dat onder een ander element valt. vb: div p{}. Voordeel is dat je specifieker elementen kunt targetten.
  • class selectors: hiermee kun je elementen targetten die tot een bepaalde klasse horen. vb: .form-control{}. Voordeel is dat je beter elementen kunt uitsluiten waar je de styling niet op wilt toepassen.
