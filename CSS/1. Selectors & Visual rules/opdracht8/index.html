Cascade: betekent dat een volgende CSS-regel een voorgaande CSS-regel overschrijft. Voorbeeld:

h1 { color: red; }
h1 { color: blue; }

Gevolg is dat de laatste regel van toepassing zal zijn en de h1-element blauw zal worden.

Inheritance: betekent dat sommige CSS-regels worden geërfd van hun parent-element.

Voorbeeld:

body { color: blue; }
span { color: black; }

Hier krijgt de body een blauwe kleur en wordt deze overgeërd door haar child elements, tenzij anders wordt gespecificeerd.
