Child selector: Met child selectors kun je een element targetten dat direct onder een element valt. Voorbeeld: ul.my-things > li { margin: 2em; }

Adjacent sibling selector: selecteert een element dat een specifiek element opvolgt en wanneer beide element een gezamelijke parent hebben. Voorbeeld: img + p { font-weight: bold; }

General sibling selector: doet het voorgaande, maar heeft niet als vereiste dat hij het eerst geselecteerde element moet opvolgen. Voorbeeld: img ~ p { color: red; }
